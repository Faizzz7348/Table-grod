üíª Basic Usage

Cara paling sederhana menggunakan ImageLightbox dengan array of image URLs:

import ImageLightbox from './components/ImageLightbox'; function MyComponent() { const images = [ 'https://example.com/image1.jpg', 'https://example.com/image2.jpg', 'https://example.com/image3.jpg' ]; return ( <ImageLightbox images={images} rowId="gallery-1" /> ); }

üöÄ Advanced Usage with Captions

Gunakan objects untuk menambahkan caption dan description pada setiap gambar:

const images = [ { url: 'https://example.com/image1.jpg', caption: 'Beautiful Sunset', description: 'A stunning view of the sunset over the ocean' }, { url: 'https://example.com/image2.jpg', caption: 'Mountain Peak', description: 'Snow-capped mountains at dawn' } ]; <ImageLightbox images={images} rowId="gallery-2" />

üìä Integration with DataTable

Contoh penggunaan dalam PrimeReact DataTable:
